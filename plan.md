# Growth Hub - Education & Opportunity Platform for Youth

A modern, responsive platform dedicated to empowering young people with education, scholarships, and career opportunities.

## Tech Stack
- React 19 + Vite
- Tailwind CSS
- Framer Motion (Animations)
- Lucide React (Icons)
- Shadcn UI (Components)

## Feature Breakdown
1. **Landing Page**
   - Animated Hero section with clear CTA.
   - Dynamic Statistics section (Impact tracking).
   - "Featured Opportunities" carousel/grid.
   - "Top Learning Paths" grid.
   - Mentorship & Community highlights.
2. **Opportunities Page**
   - Filterable list of scholarships, internships, and volunteer roles.
3. **Education Platform**
   - Course categories and preview cards.
4. **Responsive UI**
   - Mobile-first navigation with slide-out menu.
   - Smooth scroll and reveal animations.
